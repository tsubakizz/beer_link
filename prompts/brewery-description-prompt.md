# ブルワリー説明文生成プロンプト

## 目的

クラフトビールファンサイト（Beer Link）に掲載するブルワリー紹介文を作成する。

## 手順

### Step 1: 基本情報の取得

https://beer-link.com/breweries/[N] にアクセスし、以下を取得:
- ブルワリー名
- 「Website:」に表示されている公式サイトURL

### Step 2: Web検索による情報収集

「[ブルワリー名] クラフトビール ブルワリー」で検索し、以下の情報源を確認:

**優先度高（信頼性の高いソース）:**
- 公式サイト
- ビールの縁側（beer-engawa.jp）- ブルワーインタビューが豊富
- たのしいお酒.jp（tanoshiiosake.jp）- 詳細な紹介記事
- My CRAFT BEER（mycraftbeers.com）- ブルワリー情報

**優先度中:**
- 地域のニュース記事
- ブルワリー開業時のプレスリリース
- イベント出店時の紹介記事

### Step 3: 情報の収集

各ソースから以下を収集:

1. **コンセプト・スローガン**: サイトに明記されている場合のみ（原文ママで記録）
2. **ブルワー/創業者の経歴**: 前職、きっかけ、想い
3. **名前の由来**: ブルワリー名に込められた意味
4. **創業年**: 正確な年月
5. **住所**: 醸造所の所在地（都道府県・市区町村・番地）
6. **特徴・こだわり**: 醸造方法、使用原料、独自の取り組み
7. **代表銘柄**: 看板商品、受賞歴
8. **ロゴ画像URL**: 公式サイトから取得

### Step 4: 説明文の作成

収集した情報のみを使って説明文を作成する。

## 制約事項

### 情報の正確性

- 収集したソースに記載されている情報のみを使用すること
- 独自の解釈や、一般的・補足的な情報の追加（「職人が丹精込めて」などの推測）は一切禁止
- コンセプト、スローガン、商品名は、サイト内の表記（原文ママ）を維持し、勝手に語尾やニュアンスを改変しないこと
- 勝手にキャッチコピーなどを推測しないこと。テキストとしてコンセプト、スローガン、キャッチコピーとサイトに記載がある物を判断すること
- カギ括弧（「」）で囲まれた引用文は絶対に変えないこと

### 説明文に含めるべき要素（優先順）

1. 創業年・所在地
2. ブルワー/創業者の特徴（経歴、想いなど）
3. 名前の由来（あれば）
4. 醸造の特徴・こだわり
5. 代表銘柄・受賞歴

### 文字数

- 詳細ページ用説明文: 400文字以内
- 一覧ページ・description用説明文: 100文字以内

## 出力形式

```
BREWERY_ID: [N]
BREWERY_NAME: [ブルワリー名]
ADDRESS: [住所、見つからなければ空]
DESCRIPTION_LONG: [詳細ページ用説明文、400文字以内]
DESCRIPTION_SHORT: [一覧・description用説明文、100文字以内]
LOGO_URL: [ロゴ画像のURL、見つからなければ空]
SOURCES: [情報の根拠（どの文言をどこから引用したかをURL付きで箇条書き）]
```

## CSV出力形式

最終的には以下のCSV形式で出力する：

| カラム | 説明 |
|--------|------|
| id | ブルワリーID |
| address | 住所（無ければ空） |
| description_long | 詳細ページに載せる説明文（400文字以内） |
| description_short | 一覧ページ・description に載せる説明文（100文字以内） |
| logo_url | ロゴ画像のURL（無ければ空） |
| sources | 情報の根拠（URL付き） |

## 情報が薄い場合の対処

公式サイトの情報が薄い場合は、以下を試す:

1. 「[ブルワリー名] インタビュー」で検索
2. 「[ブルワリー名] 開業」「[ブルワリー名] オープン」で検索
3. 「[ブルワリー名] ブルワー」で検索
4. SNS（Instagram、X）の公式アカウントを確認

それでも情報が不足する場合は、取得できた情報のみで簡潔な説明文を作成する。
